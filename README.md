## 开始

1. 进入config\base_settings.py配置自己环境信息
2. 创建一个虚拟环境，然后`pip install -r requirements.txt`
3. 启动服务器，`python manage.py runserver`
4. 打开浏览器，输入`http://127.0.0.1:9999/`即可进入系统首页
